# Super-Mario
Ce TP est réaliser dans l'apprentissage du langage Scala. Il a pour but de créer un simple jeu de Mario en console.
Pour gagner, Mario doit :

sauter ;

resauter ;

battre un ennemi ;

entrer dans le château ;

vaincre Bowser ;

sauver Peach.
